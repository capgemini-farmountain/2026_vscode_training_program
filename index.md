# AI Productivity Engineering with VS Code Copilot

A self-paced training program for developers and business users.

## Choose Your Path

| Path | Audience | Modules | Time |
|------|----------|---------|------|
| [Copilot Essentials](paths/essentials.md) | Everyone | M0 → M1 | ~6h |
| [Developer AI Engineering](paths/developer.md) | Developers | M0 → M5 | ~20h |
| [Cowork & Automation](paths/cowork.md) | Business users | M0, M1, C0 → C4 | ~24h |
| [AI Engineering Complete](paths/complete.md) | Full depth | All modules | ~38h |

## Before You Start

All paths begin at **[M0 Foundation](curriculum/m0-foundation/README.md)**.
No prerequisites. Bring VS Code with GitHub Copilot installed.

## What You'll Build

Every module produces a real artifact you keep:

- Your own token economics reference (M0)
- A personal prompt library and workspace instructions (M1)
- A custom agent in your personal Superpowers fork (M2)
- An original skill you wrote yourself (M3)
- A published plugin with an install URL (M4)
- A capstone AI system you can demonstrate (M5 or C4)
